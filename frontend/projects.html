QRLizer

This Flutter-based mobile application is designed to detect QR code phishing (quishing) attacks. Users can scan QR codes, and the app extracts the embedded URL or data, which is then analyzed using a backend API to determine whether the link is safe or malicious.

Key Strokes Analyzer

A lightweight security tool that monitors and blocks unauthorized background .exe files from accessing user input. It ensures user privacy by detecting hidden keyloggers, suspicious processes in real time, and alerting users with actionable notifications.
